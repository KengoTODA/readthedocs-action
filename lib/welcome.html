Welcome to rtd-bot!

An unofficial GitHub Probot that helps reveiwing on Read The Docs (RTD)

Thanks for installation! Before you enjoy our bot, please make sure followings:

  1. Your RTD project has been connected with GitHub repository, or integrated via GitHub webhook.
  2. Invite rtd-bot user to your RTD project as maintainer.
  3. Add rtd.project config to .github/config.yml file in your repo. See example for detail.
  4. Enable rtd-bot in your repo from the rtd-bot page at GitHub.

Enjoy hacking with Read The Docs!
